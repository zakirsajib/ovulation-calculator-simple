11	== Description ==
12	
13	This Ovulation Calculator will calculate your most fertility days. You can use shortcode to page or post or sidebar. The form itself and the text results are mobile friendly and responsive, built with usability in mind. The Ovulation Calculator for WordPress Plugin can be successfully used in desktop, mobile and tablet environment.
14	
15	== Installation ==
16	
17	1. Upload the plugin directory to the `/wp-content/plugins/` directory
18	2. Activate the plugin through the 'Plugins' menu in WordPress
19	3. Go to Ovulation Calculator section in WordPress dashboard and enter details.

22	== Frequently Asked Questions ==
23	
24	= Does the calculator have shortcode ? =
25	
26	Yes. The shortcode is [ovulationcalculator]. You can use it both in post and page and even in sidebar.
27	
28	= Does this plugin support MailChimp ? =
29	
30	Only in Pro version (Coming soon). You can enter your MailChimp API Key and Unique List ID in mailchimp tab of the plugin. After user fills out the email field in calendar then it will be registered to the mailchimp's list instantly.
31	
32	= Can I translate/rewrite the wordings of this plugin ? =
33	
34	In terms of localisation, you are free to change any text and / or date format, from plain English to any language, from the settings page.
35	
36	= Does this plugin translate ovulation dates according to language chosen? =
37	
38	Only in Pro version (Coming soon). It supports 7 languages. They are Danish, Swedish, Norwegian, Finnish, English, Estonian and Spanish.
